# WaliaBet - Enterprise Sports Betting Platform

You are a principal software architect and senior full-stack engineer.

Build **WaliaBet**, a modern enterprise-grade sportsbook platform with:

* Public Website
* Customer Web Application
* Telegram Bot
* Admin Dashboard
* REST API
* PostgreSQL Database
* Secure Authentication
* Responsive UI
* Dark & Light Mode

The project must be production-ready, modular, scalable, clean, and easy to maintain.
